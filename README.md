# dpl-ha

Одна страница (`index.html`) для GitHub Pages.

## Использование

Открывайте только корневую ссылку:

`https://<ваш-домен>/?url_ha=<URL_ПОДПИСКИ>`

Пример:

`https://<ваш-домен>/?url_ha=https%3A%2F%2Fpanel.dagdev.ru%2Fsub%2Fxxxx`

## Что есть на странице

1. Вкладки приложений: **Happ** и **Hiddify**
2. Кнопки установки для iPhone / Android / Windows / MacBook
3. Кнопка импорта подписки:
   - Happ: `happ://add/<URL_ПОДПИСКИ>`
   - Hiddify: `hiddify://add/<URL_ПОДПИСКИ>`
4. Кнопка техподдержки: `https://t.me/kh4ck`
