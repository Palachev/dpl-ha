# dpl-ha

Одна страница (`index.html`) для GitHub Pages.

## Использование

Открывайте только корневую ссылку:

`https://<ваш-домен>/?url_ha=<URL_ПОДПИСКИ>`

Пример:

`https://<ваш-домен>/?url_ha=https%3A%2F%2Fpanel.dagdev.ru%2Fsub%2Fxxxx`

Страница показывает:
1. Кнопку «Скачать Happ»
2. Кнопку «Добавить подписку» (deeplink `happ://add/<URL_ПОДПИСКИ>`)
3. Второй способ: «Скопировать ссылку подписки» и импорт через буфер обмена в Happ:
   - открыть Happ
   - нажать `+` в правом верхнем углу
   - выбрать «Вставить из буфера обмена»
